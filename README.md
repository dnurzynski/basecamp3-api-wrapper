# Basecamp3 API Wrapper

A simple Ruby Wrapper for the Basecamp 3 API

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'basecamp3'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install basecamp3
